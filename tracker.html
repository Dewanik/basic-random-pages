How to track your pages

To allow this dashboard to see the data, you must add a lightweight tracking script to the bottom of the pages you want to monitor (e.g., bakery.html, test.html).

Copy and paste this right above the closing </body> tag in your HTML files:

<script>
  // Automatically detects the filename (e.g. "bakery.html") and increments its hit count
  const pageName = window.location.pathname.split("/").pop() || "index.html";
  fetch(`https://api.counterapi.dev/v1/Dewanik-basic-random-pages/${pageName}/up`)
    .catch(err => console.error("Hit tracker error:", err));
</script>
